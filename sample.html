Simple jQuery Accrodion

Can be used in sidebar as a nav menu, or in your main content.

All style can be changed, except for .accordion-panel this needs to be set as display:none;.

jQuery Code:

$('document').ready(function(){
$('.accordion').on('click', '.accordion-control', function(e){ e.preventDefault();
$('.accordion-panel').not($(this)).slideUp();
$(this).next('.accordion-panel').not(':animated').slideToggle();
}); });
